Currency growth app endpoints

/currency-growth?currencyCode={currency}

If {currency} increased compared to yesterday then random rich gif is returned http://api.giphy.com/v1/gifs/random?tag=rich

If {currency} decreased compared to yesterday then random broke gif is returned http://api.giphy.com/v1/gifs/random?tag=broke

If {currency} entered does not exist then error page is returned
